# meta/ — Мета-данные о Telegram-каналах

Эта папка содержит промежуточные и итоговые файлы с информацией о Telegram-каналах, полученные из датасета `ajtkulov/telegram-ru` на Hugging Face.

Все файлы созданы в процессе очистки, фильтрации и нормализации данных для дальнейшего отбора постов по нишевым категориям (без новостей, политики, шок-контента, эротики и т.п.).

## Файлы и их назначение

- **`all.channel.csv`**  
  Исходный файл, скачанный и распакованный из ZIP-архива датасета.  
  Самый сырой вариант — не используется дальше.

- **`all_channels_clean.csv`**  
  Очищенный список каналов: первый столбец приведён к чистому username (без `https://tgstat.ru/channel/` и `@`).  
  **Самый важный и актуальный файл** для создания словаря `channel → category` и фильтрации постов.

- **`unique_categories.csv`**  
  Список всех уникальных категорий каналов (по одной категории на строку).  
  Получен после чтения очищенного списка каналов.  
  Полезен для анализа и ручной проверки категорий.

- **`filtered_categories.csv`**  
  Отфильтрованный список категорий: удалены новости, политика, шок-контент, эротика, Darknet, религия и другие нежелательные тематики.  
  **Основной фильтр** для отбора каналов и постов.

## Основной рабочий файл

**`all_channels_clean.csv`** — это то, что используется в коде для сопоставления каналов с категориями.

Пример загрузки в Python:

```python
all_channels_clean = pd.read_csv("meta/all_channels_clean.csv")
filtered_categories = pd.read_csv("meta/filtered_categories.csv")

allowed_categories = set(filtered_categories['category'].dropna().astype(str))
channel_to_category = dict(zip(all_channels_clean['link'], all_channels_clean['category']))
```
